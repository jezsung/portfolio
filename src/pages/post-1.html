HTML and CSS Basics

What is HTML?

Hyper Text Markup Language(HTML) is a markup language for building user interfaces of websites.

What are HTML tags?

Tags mark the start and the end of an HTML element. Each tag has its own name and the name is enclosed by angle brackets. For example, the h1 tag marks the start with enclosing angle brackets, h1, and marks the end with enclosing angle brackets and a backslash, h1. If a backslash comes after the name of the tag—such as br and hr—, the tag is a self-closing tag and it does not require a separate closing tag.

What are HTML elements?

HTML elements consist of tags, attributes, and optional content inside of a start tag and an end tag. For example, an h1 element could look like below. 

<h1 class="title">Hello World!</h1>

The element has a start and an end tag, a class attribute with the value title, and the content Hello World! inside of the tags. Browsers show HTML elements with predefined formatting standards.

What are HTML attributes?

HTML attributes change the behavior of certain elements or work as an identifier to be referenced in CSS and Javascript.

What are inline elements?

Inline elements do not shift the content after the elements. Inline elements are useful when you only want to style a certain part of the text. For example, the i element can make a text style italic without introducing a new line. 

<div>
Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatem eligendi blanditiis sequi nam vel, pariatur maiores minima ad! Libero quasi odit sit maiores vel ipsam rerum eveniet soluta aut porro.
</div>

What are block elements?

As opposed to inline elements, block elements introduce a new line. The most commonly used block element is the div element. If a div element gets inserted after another div element, the content of the former div element will appear in a new line instead of being inserted after the content of the latter div element. 

<div>Hello World!</div>
<div>This will appear in a new line.</div>

What is CSS rule?

Cascading Style Sheets(CSS) styles the linked HTML document by sets of properties, and values and applies them to the elements selected by selectors.

What are CSS selectors?

CSS selectors apply a set of properties to only the selected HTML elements. CSS selectors can select HTML elements by the name of the class, id, and relationships with other elements.

What are CSS values?

CSS values are applied to CSS properties and adjust the color values, amount of padding, etc.
